# Third-party posture

TsaoSciComputation is independently implemented under MIT. It references public scientific software only for interoperability and method vocabulary. No third-party source code, executable, license key, pseudopotential, basis database, force field, or copyrighted manual is vendored.

External tools retain their own licenses and citation requirements. Users must verify lawful access and cite the exact versions and methods actually used. Commercial names such as Gaussian, VASP, and Aspen identify optional integration targets, not bundled software or verified availability.
