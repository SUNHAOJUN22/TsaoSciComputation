# TsaoSciComputation

Use this skill to turn a scientific question into a traceable computation program across electronic, atomistic, mesoscale, continuum, reactor, process, control, and digital-twin scales.

1. Create a calculation contract before choosing software.
2. Route by scale, observable, method fitness, and available lawful environment.
3. Probe tools; never pretend a solver or license is available.
4. Separate `completed`, `parsed`, `converged`, `validated`, and `accepted`.
5. Record numerical convergence, physical checks, uncertainty, applicability, evidence, and approvals.
6. Use structured multiscale handoffs with units, conditions, reference states, transformations, and uncertainty.
7. Escalate model, license, safety, and unknown failures to a human gate.
