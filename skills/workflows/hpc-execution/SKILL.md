# HPC and reproducible execution

Route a scientific request to `hpc-execution` only when its scale, observables, boundary conditions, and evidence requirements are explicit. A completed process is not automatically converged, validated, or accepted.

Required gates: contract, preflight, completion, convergence, physical_validation, uncertainty, acceptance.
